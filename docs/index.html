Rename It

Keep your Sketch files orgnized.
Batch renaming layers and artboards has never been so easy.

Install Plugin Version: 3.0

Rename in Sequence

Sequencelly rename layers in either ascending or descending order.

Keywords

  • %n Asceding numbered sequence
  • %N Descending numbered sequence
  • %A Alphabet sequence

Pro Tip

  • %nn This will output 01, 02, 03 and so on
  • %a Lowercase alphabet sequence

Current Layer Name

The keyword * will copy the current selected layer(s) name.

Pro Tip

  • If you need to use "*" as text, you can escape it like so '*
